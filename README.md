웹 비디오 녹화 어플리케이션.<br>


## 주요 기능
- 카메라 녹화
- 화면 녹화
- 녹화된 영상 다운로드

## 활용 가능 용도
- 화상 회의 녹화
- UX 시나리오 재현
- 가이드 영상 제작

## 기술 스텍
- Typescript
- React
- Recoil(상태관리 라이브러리)

## 명령어
- 서버 실행: ```yarn start```
- 빌드: ```yarn build```
- 스토리 북 실행: ```yarn storybook```
